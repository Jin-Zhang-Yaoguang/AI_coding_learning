# kmTravel · 设计路线 Prompt 存档

> 用于生成 `kmTravel.html`（昆明 4 日游旅行攻略页面）的原始需求 Prompt，便于复用与迭代。

## 原始 Prompt

```text
我五一计划去昆明游玩 4 天的旅行攻略。

需求：
1. 帮制作旅行攻略，考虑出行时间和路线，以及天气状况路线规划。
2. 制作网页地图自定义绘制旅游路线和位置。
3. 网页使用简约美观页面风格，景区图片以卡片展示。
4. 行程规划结果在高德地图 App 展示，并集成到 H5 页面中。
5. 同一天行程景区之间我想打车前往。
6. 生成文件名 kmTravel.html，生成的 html 存储在创建的 mcp test 文件夹下。
```

## 需求拆解（执行要点）

- **目的地与时长**：昆明，五一假期，4 天。
- **行程规划**：考虑出行时间、路线就近排序、五一天气（温和/午后阵雨/紫外线强）。
- **地图能力**：网页内高德地图，自定义绘制路线（连线）和景点位置（标记）。
- **页面风格**：简约美观，景区图片以卡片展示。
- **App 集成**：行程通过高德地图 App 展示（URI 唤起导航/定位），并嵌入 H5 页面。
- **同日交通**：同一天景点之间打车（驾车 mode=car）前往。
- **产物**：`kmTravel.html`，存放于 `mcp test/` 文件夹。

## 技术实现备注

- 高德 JS API v2.0 加载地图，`AMap.Marker` 标点 + `AMap.Polyline` 自定义绘制当日行车顺序连线，并尝试 `AMap.Driving` 叠加真实道路路线（失败回退连线）。
- 高德 URI API 唤起 App：
  - 单段打车导航：`https://uri.amap.com/navigation?from=...&to=...&mode=car`
  - 单点定位：`https://uri.amap.com/marker?position=...`
- key 取自项目 `.cursor/mcp.json`；如需真实驾车路线稳定显示，需补充 JS Key + 安全密钥 `securityJsCode`。
